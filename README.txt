Bonjour !

La scène principale se nomme MainScene et se trouve dans ProjetRTS2 -> Assets -> Projet -> Scenes -> MainScene
Il n'y a qu'une seule branche sur le projet actuellement

Le projet est jouable uniquement à la souris
Clic gauche sert à selectionner un agent
Maintenir le clic gauche permet de dessiner un carré de selection qui peut selectionner plusieurs unités à la fois
Clic droit sert à donner des ordres à ses unités après les avoir selectionnées

Debug commandes : elles sont desactivées par défaut et nécessitent d'appuyer une fois sur P pour les activer
- A : rajouter 100 énergie
- Z : retirer 100 énergie
- Q : diminuer le timeScale par 0.2
- S : rétablir le timeScale à 1
- D : augmenter le timeScale par 1
- W : passer à la nuit (X indisponible pendant 5 sec après ça)
- X : passer au jour (W indisponible pendant 5 sec après ça)
- R : reload la scène
- Space : lock/delock la caméra
